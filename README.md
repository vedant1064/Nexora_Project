# 🚀 Autonex AI

AI-powered WhatsApp Business Automation SaaS  
Built with FastAPI + OpenAI + React + PostgreSQL + Razorpay

---

## 📌 Overview

Autonex AI is a multi-tier WhatsApp automation platform designed for modern businesses and MSMEs.

It enables:

- Smart automated replies
- AI-driven product responses
- Tier-based AI behavior (Starter, Pro, Premium)
- Lead classification & scoring
- Sales funnel analytics
- Conversation history tracking
- Subscription-based SaaS billing

---

## 🏗️ Architecture

### Frontend
- React (Dashboard UI)
- Tailwind CSS
- Lucide React Icons

### Backend
- FastAPI
- OpenAI API (LLM)
- Razorpay Subscription API
- PostgreSQL
- JSON-structured AI output

### Database Tables
- `businesses`
- `products`
- `leads`
- `conversation_history`

## 💎 Subscription Tiers

### 🟢 Starter (₹499/month)
- Smart Replies
- Product Information
- Hindi/English Support
- Basic Automation
- Max 4 sentences
- No memory

### 🔵 Pro (₹999/month)
- Quantity Detection
- Total Price Calculation
- 1 Lead Qualification Question
- Session Memory
- Basic Analytics
- Max 6 sentences

### 🟣 Premium (₹1999/month)
- Full Conversation Memory
- Lead Classification (Hot/Warm/Cold)
- Conversion Probability (0–100%)
- Objection Handling
- Upselling Logic
- Lead Intelligence Panel
- Max 8 sentences
